## 🚀 Features

- **Responsive Design:** Works on desktop and mobile devices
- **Hero Section:** Introduction with profile image in a circular frame
- **About Section:** Personal summary and current focus
- **Certificates Section:** Showcase of certificates with images
- **Projects Section:** Project cards with toggleable overview details
- **Clean UI:** Modern design with smooth animations

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Styling:** Custom CSS with responsive grid layouts
- **Interactivity:** Vanilla JavaScript for DOM manipulation

## 📖 Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/SantoshDhanvade/Profile.git
   ```

2. Open `index.html` in your browser to view the profile page.

3. Add your own images:
   - Place your profile image in `Profile Img/profile.jpg`
   - Add certificate/project images in `Certificate/` folder
